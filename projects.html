Projects

StuddiBuddi

This was a hackathon project from YHacks. We came up with the idea when considering how it is sometimes hard to find people to study with. The goal of the service was to allow users to create study hotspots for a subject and time. Other users would then be able to search and join study hotspots to encourage group studying.

Python Game Suite

This was a school project where we used pygame and tkinter to make a set of mini-games. All our code is stored on Github, so feel free to check it out!
